# Comics

Comics is a Godot 4 project for an Android-tablet-first comic staging app. The core workflow is to create comic projects, pose simple placeholder characters and objects in reusable scenes, capture camera-based ComicFrames, and arrange those frames into PageLayouts for export.

## Current Status

App shell, project storage, minimal Home, and minimal Workspace are implemented. The original camera and dummy-pose prototype is preserved for later migration into frame editing.
